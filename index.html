About Asphalt Yield Calculator

Overview

The Asphalt Yield Calculator is a specialized tool designed for asphalt paving professionals to calculate and track material usage, paving speed, projected lengths, and tonnage requirements.

How to Use

Yield Calculator

  1. Enter the Load Number for identification
  2. Input Truck Tonnage in tons
  3. Enter Start and End Station values
  4. Input Width in feet
  5. The Length is calculated automatically
  6. Click "Save Truck Data" to record the entry
  7. View results in the Current Truck Results and Day Summary sections

Data History

View all saved truck data for the current day, with totals at the bottom. Click "View Previous Days" to access archived data from past days.

Paver Speed Calculator

Calculate optimal paver speed by entering Production Rate (tons/hour), Pavement Width (feet), and Application Rate (lbs/sq yd).

Projected Length Calculator

Determine projected paving length based on Projected Daily Tons, Lane Width, and Application Rate.

Tonnage Calculator

Estimate the amount of HMA (Hot Mix Asphalt) needed based on Length, Width, and Depth measurements.

Data Management

  • Click "Start New Day" to archive current data and begin a new day
  • Export data to CSV format for further analysis using the "Export All Data" option
  • All data is stored locally on your device
